## [0.0.1] - 2026-04-19

### Added
- Initial release of AI JSON Optimizer
- Smart JSON to TOON conversion with k/d format
- Recursive handling of nested objects and arrays
- Token estimation and savings calculation
- Automatic format selection (JSON vs TOON) based on token efficiency
- In-place editor replacement
- Clipboard copy functionality
- Minified output by default
- VS Code settings integration
- Comprehensive error handling

### Features
- Optimize JSON for AI command
- Intelligent optimization logic for arrays and large objects
- Token reduction notifications
- Support for complex nested structures
